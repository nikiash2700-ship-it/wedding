# 🤖 Инструкция по настройке Telegram бота

Эта инструкция поможет вам создать Telegram бота для получения анкет гостей.

## 📋 Шаг 1: Создание бота

1. Откройте Telegram и найдите бота **@BotFather**
   - Нажмите "Start" или отправьте команду `/start`

2. Создайте нового бота:
   - Отправьте команду `/newbot`
   - Введите имя для вашего бота (например: "Свадьба Алины и Ивана")
   - Введите username для бота (должен заканчиваться на `bot`, например: `wedding_2024_bot`)

3. **Скопируйте токен бота** - он будет выглядеть примерно так:
   ```
   1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
   ```
   ⚠️ **ВАЖНО:** Сохраните этот токен! Он понадобится для настройки сайта.

---

## 📋 Шаг 2: Получение вашего Chat ID

Chat ID - это ваш личный идентификатор в Telegram, куда будут приходить анкеты.

### Способ 1: Через бота @userinfobot (самый простой)

1. Найдите в Telegram бота **@userinfobot**
2. Нажмите "Start"
3. Бот отправит вам вашу информацию, включая **ID**
4. Скопируйте это число (например: `123456789`)

### Способ 2: Через бота @getidsbot

1. Найдите бота **@getidsbot**
2. Нажмите "Start"
3. Бот покажет ваш Chat ID

### Способ 3: Через вашего бота

1. Напишите вашему боту любое сообщение (например: "Привет")
2. Откройте в браузере:
   ```
   https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
   ```
   (Замените `ВАШ_ТОКЕН` на токен из шага 1)
3. Найдите в ответе `"chat":{"id":123456789}` - это ваш Chat ID

---

## 📋 Шаг 3: Настройка сайта

1. Откройте файл `script.js` в текстовом редакторе (Блокнот, Notepad++ и т.д.)

2. Найдите строки (примерно строки 72-73):
   ```javascript
   const TELEGRAM_BOT_TOKEN = 'YOUR_BOT_TOKEN';
   const TELEGRAM_CHAT_ID = 'YOUR_CHAT_ID';
   ```

3. Замените значения:
   ```javascript
   const TELEGRAM_BOT_TOKEN = '1234567890:ABCdefGHIjklMNOpqrsTUVwxyz'; // Ваш токен
   const TELEGRAM_CHAT_ID = '123456789'; // Ваш Chat ID
   ```

4. Сохраните файл

---

## 📋 Шаг 4: Проверка работы

1. Откройте сайт в браузере
2. Заполните тестовую анкету
3. Нажмите "Отправить анкету"
4. Проверьте Telegram - вы должны получить сообщение с данными анкеты!

## ❓ Часто задаваемые вопросы

**Q: Бот не отвечает на сообщения?**  
A: Это нормально! Бот создан только для получения анкет с сайта. Он не отвечает на сообщения.

**Q: Можно ли получать анкеты нескольким людям?**  
A: Да! Можно создать группу в Telegram, добавить туда бота, и указать Chat ID группы. Или использовать несколько Chat ID (потребуется небольшая доработка кода).

**Q: Безопасно ли хранить токен в коде?**  
A: Для личного сайта-приглашения это нормально. Токен виден в коде сайта, но это не критично для вашего случая. Если хотите большей безопасности, можно использовать серверную часть.

**Q: Как получить Chat ID группы?**  
A: Добавьте бота в группу, напишите в группе любое сообщение, затем откройте:
```
https://api.telegram.org/botВАШ_ТОКЕН/getUpdates
```
Найдите `"chat":{"id":-123456789}` - отрицательное число это Chat ID группы.

## 🔧 Дополнительные настройки (опционально)

### Настройка команд бота

Если хотите, чтобы бот отвечал на команды:

1. Напишите @BotFather команду `/setcommands`
2. Выберите вашего бота
3. Добавьте команды, например:
   ```
   start - Начать работу
   help - Помощь
   ```

### Изменение описания бота

1. Напишите @BotFather команду `/setdescription`
2. Выберите вашего бота
3. Введите описание, например: "Бот для сбора анкет гостей на свадьбу"

---

## 🆘 Если что-то не работает

1. **Проверьте токен** - убедитесь, что скопировали его полностью
2. **Проверьте Chat ID** - должно быть число (может быть отрицательным для групп)
3. **Откройте консоль браузера** (F12) и посмотрите на ошибки
4. **Проверьте интернет-соединение**
5. Убедитесь, что бот не заблокирован
6. **Убедитесь, что сайт открывается через HTTPS** (не file://) - для этого используйте GitHub Pages или другой хостинг
