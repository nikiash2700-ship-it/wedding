# 🔧 Исправление ошибки 404 на GitHub Pages

## ⚠️ Проблема: 404 File not found

Это значит, что GitHub Pages не может найти файл `index.html` в репозитории.

---

## ✅ Решение: Проверьте, что файлы загружены

### Шаг 1: Проверьте репозиторий

1. Откройте: https://github.com/nikiash2700-ship-it/wedding
2. **Проверьте, есть ли файлы:**
   - Должен быть файл `index.html`
   - Должен быть файл `styles.css`
   - Должен быть файл `script.js`
   - И другие файлы

### Если файлов НЕТ:

**Загрузите файлы:**

1. На странице репозитория нажмите **"uploading an existing file"**
2. Перетащите все файлы из папки `C:\Users\Алина\свадьба\`:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
   - И другие файлы
3. Внизу страницы:
   - **Commit message:** `Add website files`
   - Нажмите **"Commit changes"**

### Если файлы ЕСТЬ, но в подпапке:

**Проверьте структуру:**
- Файлы должны быть в **корне** репозитория (не в подпапке)
- Если файлы в папке (например, `свадьба/index.html`), нужно:
  - Либо переместить файлы в корень
  - Либо в настройках Pages выбрать папку `/свадьба`

---

## ✅ Шаг 2: Проверьте настройки GitHub Pages

1. Откройте: https://github.com/nikiash2700-ship-it/wedding/settings/pages
2. Проверьте настройки:
   - **Source:** Branch `main`
   - **Folder:** `/` (root) - если файлы в корне
   - Или `/свадьба` - если файлы в папке `свадьба`
3. Нажмите **"Save"**

---

## ✅ Шаг 3: Подождите

После загрузки файлов или изменения настроек:
- Подождите **1-2 минуты**
- GitHub Pages обновляется не мгновенно
- Обновите страницу сайта (F5)

---

## 🔍 Проверка структуры репозитория

**Правильная структура:**
```
wedding/
  ├── index.html  ← должен быть в корне!
  ├── styles.css
  ├── script.js
  ├── README.md
  └── другие файлы...
```

**Неправильная структура:**
```
wedding/
  └── свадьба/
      ├── index.html  ← файлы в подпапке
      └── ...
```

---

## 💡 Быстрое решение через GitHub Desktop

Если используете GitHub Desktop:

1. Откройте GitHub Desktop
2. Убедитесь, что репозиторий `wedding` подключен
3. Проверьте, что файлы в папке репозитория
4. Если файлов нет - скопируйте их туда
5. Сделайте Commit и Push

---

1. Подождите 1-2 минуты
2. Обновите страницу сайта: https://nikiash2700-ship-it.github.io/wedding/
3. Сайт должен заработать!
