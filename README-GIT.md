# Автоматическая загрузка файлов на GitHub

Этот проект настроен для автоматической загрузки изменений на GitHub.

## 🚀 Быстрый старт

### Вариант 1: Автоматическая загрузка после каждого коммита (РЕКОМЕНДУЕТСЯ)

1. **Откройте папку проекта** в терминале (или в Cursor)

2. **Запустите скрипт настройки:**
   ```powershell
   .\setup-auto-push.ps1
   ```

3. **Готово!** Теперь после каждого коммита изменения будут автоматически загружаться на GitHub.

**Как использовать:**
```powershell
# 1. Добавьте изменения
git add .

# 2. Создайте коммит (изменения автоматически загрузятся!)
git commit -m "Описание изменений"
```

### Вариант 2: Ручная загрузка одним скриптом

Если хотите контролировать, когда загружать:

```powershell
# Просто запустите скрипт в папке проекта
.\auto-push.ps1

# Или с вашим сообщением коммита:
.\auto-push.ps1 -CommitMessage "Обновлен дизайн"
```

## 📝 Обычная работа с Git

Если автоматизация не нужна, используйте стандартные команды:

```powershell
# 1. Посмотреть изменения
git status

# 2. Добавить все изменения
git add .

# 3. Создать коммит
git commit -m "Описание ваших изменений"

# 4. Загрузить на GitHub
git push origin main
```

## 🔧 Настройка в GitHub Desktop

Если используете GitHub Desktop:

1. Внесите изменения в файлы
2. В GitHub Desktop появятся изменения в разделе "Changes"
3. Введите описание коммита
4. Нажмите "Commit to main"
5. Нажмите "Push origin" (или включите автоматический push в настройках)

## ⚙️ Настройка автоматического push в GitHub Desktop

1. Откройте **File → Options → Git**
2. Включите опцию **"Automatically sync after committing"** (если доступна)
3. Или используйте скрипт `setup-auto-push.ps1` для настройки Git hooks

## ❓ Частые вопросы

**Q: Скрипт не запускается?**  
A: Возможно, нужно разрешить выполнение скриптов. Запустите PowerShell от имени администратора и выполните:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

**Q: Как отключить автоматическую загрузку?**  
A: Удалите файл `.git\hooks\post-commit`

**Q: Где находится папка проекта?**  
A: В GitHub Desktop нажмите "Show in Explorer" (Ctrl+Shift+F) или "Open in Visual Studio Code" (Ctrl+Shift+A)
