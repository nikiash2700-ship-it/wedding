# Сайт-приглашение на свадьбу 💍

Красивый современный сайт-приглашение на свадьбу с адаптивным дизайном и интерактивными элементами.

## 🚀 Как использовать

1. **Откройте файл `index.html`** в любом браузере (Chrome, Firefox, Edge и т.д.)
   - Просто дважды кликните на файл `index.html`
   - Или правой кнопкой мыши → "Открыть с помощью" → выберите браузер

2. **Никаких дополнительных программ не требуется!** Сайт работает сразу.

## ✏️ Как настроить под себя

### 1. Изменить имена жениха и невесты

Откройте файл `index.html` и найдите строки:
```html
<span class="name">Алина</span>
<span class="ampersand">&</span>
<span class="name">Имя</span>
```

Замените "Алина" и "Имя" на ваши имена.

### 2. Изменить дату свадьбы

**В файле `index.html`:**
```html
<p class="hero-date">15 Июня 2024</p>
```

**В файле `script.js` (строка 2):**
```javascript
const weddingDate = new Date(2024, 5, 15, 15, 0);
// Формат: new Date(год, месяц-1, день, час, минута)
// Внимание: месяц начинается с 0 (0=январь, 5=июнь, 11=декабрь)
```

### 3. Изменить место проведения

В файле `index.html` найдите секцию "Детали свадьбы" и измените:
```html
<p>Название ресторана/зала</p>
<p class="detail-address">Адрес места проведения</p>
```

### 4. Изменить время и программу

В секции "Программа дня" измените время и описание:
```html
<div class="timeline-time">15:00</div>
<div class="timeline-content">
    <h3>Церемония</h3>
    <p>Торжественная регистрация брака</p>
</div>
```

### 5. Добавить фотографии

В секции "Галерея" замените плейсхолдеры на ваши фотографии:

**Вариант 1 - Локальные фото:**
```html
<div class="gallery-item">
    <img src="фото1.jpg" alt="Наше фото">
</div>
```

**Вариант 2 - Фото из интернета:**
```html
<div class="gallery-item">
    <img src="https://ваша-ссылка-на-фото.jpg" alt="Наше фото">
</div>
```

**Важно:** Если используете локальные фото, поместите их в ту же папку, что и `index.html`, или укажите правильный путь к папке с фото.

### 6. Изменить контакты

В секции "Контакты" измените имена и телефоны:
```html
<h3>Невеста</h3>
<p>Алина</p>
<p><a href="tel:+79001234567">+7 (900) 123-45-67</a></p>
```

### 7. Изменить цвета (опционально)

В файле `styles.css` в начале файла найдите раздел `:root` и измените цвета:
```css
:root {
    --primary-color: #d4a574;    /* Основной цвет */
    --secondary-color: #8b6f47;  /* Вторичный цвет */
    --accent-color: #f5e6d3;     /* Акцентный цвет */
    --gold: #d4af37;             /* Золотой цвет */
}
```

## 📧 Настройка отправки анкет в Telegram

Сайт настроен для отправки анкет гостей прямо в ваш Telegram! Это самый удобный способ получать все ответы.

### Быстрая настройка:

1. **Создайте Telegram бота** через @BotFather (подробная инструкция в файле `TELEGRAM_BOT_SETUP.md`)
2. **Получите токен бота** и ваш Chat ID
3. **Откройте файл `script.js`** и найдите строки:
   ```javascript
   const TELEGRAM_BOT_TOKEN = 'YOUR_BOT_TOKEN';
   const TELEGRAM_CHAT_ID = 'YOUR_CHAT_ID';
   ```
4. **Замените** на ваши данные:
   ```javascript
   const TELEGRAM_BOT_TOKEN = '1234567890:ABCdefGHIjklMNOpqrsTUVwxyz';
   const TELEGRAM_CHAT_ID = '123456789';
   ```
5. **Готово!** Теперь все анкеты будут приходить вам в Telegram

📖 **Подробная инструкция:** Смотрите файл `TELEGRAM_BOT_SETUP.md`

## 📱 Публикация в интернете

### 🌐 GitHub Pages (РЕКОМЕНДУЕТСЯ)

1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы в репозиторий
3. Включите GitHub Pages:
   - Settings → Pages
   - Source: Branch `main`
   - Folder: `/` (root)
   - Save
4. Ваш сайт будет доступен по адресу: `https://ваш-username.github.io/название-репозитория/`

**Плюсы:** 
- ✅ Решает проблему CORS (сайт работает через HTTPS)
- ✅ Бесплатно, можно обновлять сайт
- ✅ Telegram бот будет работать без ошибок

### 🌐 Netlify Drop (САМЫЙ ПРОСТОЙ)

1. Откройте: https://app.netlify.com/drop
2. Перетащите все файлы в окно браузера
3. Получите ссылку на сайт

**Плюсы:** Не нужна регистрация, работает сразу, бесплатно

## ❓ Часто задаваемые вопросы

**Q: Нужно ли устанавливать что-то дополнительно?**  
A: Нет! Просто откройте `index.html` в браузере.

**Q: Как изменить фон главной страницы?**  
A: В файле `styles.css` найдите `.hero` и измените `background`. Можно добавить свою фотографию:
```css
background: linear-gradient(...), url('ваше-фото.jpg') no-repeat center;
background-size: cover;
```

**Q: Как добавить больше фотографий в галерею?**  
A: Просто скопируйте блок `.gallery-item` и вставьте еще несколько раз с разными фото.

**Q: Работает ли сайт без интернета?**  
A: Да, полностью работает! Только шрифты Google Fonts загружаются из интернета, но сайт будет работать и без них.

**Q: Форма отправляется долго?**  
A: Форма оптимизирована для быстрой отправки. Успех показывается через 0.5 секунды, а отправка в Telegram происходит в фоне.
