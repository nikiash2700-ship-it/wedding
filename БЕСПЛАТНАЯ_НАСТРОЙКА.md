# 💰 Бесплатная настройка GitHub Pages

## ✅ GitHub Pages - ПОЛНОСТЬЮ БЕСПЛАТНО!

Сообщение о GitHub Enterprise - это просто реклама. **Обычный GitHub Pages абсолютно бесплатный!**

---

## 🚀 Бесплатная настройка (3 шага)

### Шаг 1: Создайте репозиторий

1. Откройте: https://github.com/new
2. **Repository name:** `wedding` (или другое название)
3. **Visibility:** 
   - ✅ Выберите **Public** (публичный) - это БЕСПЛАТНО
   - ❌ НЕ выбирайте Private (приватный) - для него нужен платный план
4. **НЕ отмечайте:**
   - ❌ Add a README file
   - ❌ Add .gitignore
   - ❌ Choose a license
5. Нажмите **"Create repository"**

---

### Шаг 2: Загрузите файлы

1. На странице репозитория нажмите **"uploading an existing file"**
2. Перетащите все файлы из папки `C:\Users\Алина\свадьба\`:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
   - `TELEGRAM_BOT_SETUP.md`
   - `.gitignore`
3. Внизу страницы:
   - **Commit message:** `Initial commit`
   - Нажмите **"Commit changes"**

---

### Шаг 3: Включите GitHub Pages (БЕСПЛАТНО!)

1. В репозитории нажмите **"Settings"** (вверху справа)
2. В левом меню найдите **"Pages"**
3. В разделе **"Source":**
   - **Branch:** выберите `main`
   - **Folder:** выберите `/` (root)
4. Нажмите **"Save"**

---

## ✅ Готово! Всё БЕСПЛАТНО!

Через 1-2 минуты ваш сайт будет доступен по адресу:
```
https://ваш-username.github.io/wedding/
```

**Это полностью бесплатно и навсегда!**

---

## 💡 Важно знать

- ✅ **Public репозиторий** = бесплатный GitHub Pages
- ✅ **Бесплатно навсегда** - без ограничений по времени
- ✅ **Неограниченный трафик** - сколько угодно посетителей
- ❌ **Private репозиторий** = нужен платный план (но вам это не нужно!)
