# ⚡ Быстрая загрузка в GitHub - 3 шага

## ✅ Шаг 1: Создайте репозиторий в GitHub Desktop

1. Откройте **GitHub Desktop**
2. **File → New repository**
3. Заполните:
   - **Name:** `wedding` (или `svadba`)
   - **Local path:** `C:\Users\Алина\свадьба`
   - **Description:** (можно оставить пустым)
   - **Initialize this repository with a README:** ❌ НЕ отмечайте
4. Нажмите **"Create repository"**

---

## ✅ Шаг 2: Опубликуйте репозиторий

1. В GitHub Desktop нажмите **"Publish repository"**
2. ⚠️ **ВАЖНО:** Выберите **"Public"** (публичный) - это БЕСПЛАТНО!
   - ❌ НЕ выбирайте "Private" - для него нужен платный план
3. Нажмите **"Publish repository"**

---

## ✅ Шаг 3: Включите GitHub Pages

1. Откройте ваш репозиторий на GitHub.com
2. Перейдите в **Settings → Pages**
3. В разделе **Source:**
   - Branch: `main`
   - Folder: `/` (root)
4. Нажмите **"Save"**

---
